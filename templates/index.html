推送时间设置

推送时间由 GitHub Actions 控制，默认为北京时间 23:30。

修改方法：

  1. 进入你的 GitHub 项目页面
  2. 打开 .github/workflows/daily_reminder.yml 文件
  3. 找到 cron: '30 15 * * *' 这一行
  4. 修改时间（格式：分 时，UTC时间）
  5. 北京时间 = UTC + 8（例如北京时间 21:00 = UTC 13:00）

建议：由于 GitHub Actions 存在排队现象，建议将时间设置为目标时间提前 2-2.5 小时。
